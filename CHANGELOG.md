# Changelog

All notable changes to Codex Switcher are documented here.

This project follows Keep a Changelog and Semantic Versioning.

## [1.0.0] - 2026-06-12

Initial public release.

### Added

- Desktop GUI for managing multiple Codex accounts
- One-click account switching
- Usage monitoring and refresh actions
- Warm-up controls for active accounts
- Import and export backup flows
- Tray behavior and frameless window controls
- Browser/LAN mode for serving the UI over HTTP
